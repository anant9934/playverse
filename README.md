## 🕹️ About Playverse

**Playverse** is a growing collection of 100 browser-based games built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies, no installations needed.

Open any game in your browser and start playing instantly. Each game is self-contained in a single `index.html` file, making it lightweight, portable, and fast.

## 🚀 How to Play

Each game is a **single HTML file**. To play:

1. Clone or download this repository
2. Navigate to any game folder
3. Open `index.html` in your browser

\`\`\`bash
git clone https://github.com/anant9934/playverse.git
cd playverse
# Open any game:
open "Snake Game/index.html"
open "Chess Game/index.html"
open "Tetris/index.html"
\`\`\`

Or just **[visit the live site](#)** *(coming soon)*.

## 🛠️ Tech Stack

All games are built using only:

- **HTML5** — Canvas API, DOM manipulation
- **CSS3** — Animations, transitions, responsive layouts
- **Vanilla JavaScript** — Game logic, AI, physics

## 🤝 Contributing

Want to add a game or improve an existing one?

1. Fork this repository
2. Create your feature branch: \`git checkout -b feat/new-game\`
3. Add your game in its own folder with an \`index.html\`
4. Commit your changes: \`git commit -m "feat: add [Game Name]"\`
5. Push and open a Pull Request
